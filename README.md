# 🏥 Clinic Management System

*A mobile-based solution for appointment scheduling and expense tracking*


## 🧩 Problem

Many small clinics rely on manual methods (calls, notebooks, or messaging apps) to manage appointments and expenses. This often leads to scheduling conflicts, missed appointments, and poor financial tracking.



 🎯 Objective

To build a simple and efficient system that helps clinic staff manage appointments, track expenses, and ensure patient confirmation through SMS.



 👨‍💼 My Role

* Led **requirement gathering** by interacting with clinic staff and doctor
* Identified key workflow issues in appointment handling and expense recording
* Defined **functional and non-functional requirements**
* Created **user flows and system features**
* Collaborated with the development team to align implementation with requirements


 👥 Users

* 👨‍⚕️ **Doctor** – views schedule and manages time
* 📞 **Reception/Staff** – handles appointments and expense entries


 ⚙️ Features

 📅 Appointment Management

* Create, update, and cancel appointments
* View daily schedule
* Prevent double bookings

 📲 SMS Confirmation

* Sends appointment confirmation using the device’s **default SMS app**
* Helps reduce no-shows and ensures booking validation

 💰 Expense Tracking

* Record daily expenses
* Categorize and monitor spending
* Maintain financial records



 🔄 User Workflow

1. Staff creates an appointment
2. System triggers SMS confirmation
3. Patient receives confirmation message
4. Doctor checks schedule
5. Staff logs clinic expenses



 📋 Requirements

Functional Requirements

* Manage appointments (create/update/delete)
* Send SMS confirmation after booking
* Record and view expenses
* Provide schedule access to doctor

Non-Functional Requirements

* User-friendly interface for non-technical users
* Fast and responsive performance
* Reliable data handling
* Mobile-friendly design



🧠 Key Design Decisions

* Used **SMS instead of internet-based notifications** for better accessibility
* Focused on **simplicity** for clinic staff
* Designed workflow to minimize manual errors and save time
